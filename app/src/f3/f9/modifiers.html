Модификаторы

Для чего нужны так назваемые модификаторы (modifiers) точного представления я пока не имею.

Вероятно они служат для того, чтобы изменить некие параметры юнитов, которые обычним способом через редактор изменить нельзя.

Сначала разберёмся по кнопкам. Кнопка New создаёт новый модификатор - чистое окно, как показано выше. Кнопка Open... открывает ранее сохранённый модификатор.

Кнопка Save сохраняет изменения в текущем модификаторе. Если модификатор ещё не имеет имени файла, то откроется окно выбора места как при нажатии кнопки Save as....
Кнопка Save as... позволяет сохранить текущий модификатор под другим именем.

Для добавления нового модификатора нажмите клавишу INSERT или нажмите ПКМ и выберете пункт Add..
Появятся строки дерева modifier.

В строке name указывается название модификатора.
Контекстное меню строк include и exclude содержит список из 16 элементов.

Контекстное меню строки parameters содержит список из трёх элементов.

Armor - броня, accuracy - точность, aim_range - прицельная дальность.

В качестве примера открыл модификатор из папки Дубовки (Первая миссия за СССР в БПО).

Насколько можно судить, сдесь для юнита с тегом m1 (очевидно имеется ввиду миномёт M1) прицельная дальность увеличивается в три раза.

Также модификаторы можно удалять, перемещать вверх и вниз.
