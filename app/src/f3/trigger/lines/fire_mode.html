Fire Mode

Строка fire_mode, тип lt1, устанавливает возможность юнита вести огонь. При значении hold юнит не будет стрелять без прямого приказа игрока, return - юнит стреляет только в ответ, open - юнит стреляет самостоятельно при обнаружении противника. При значении по-умолчанию возможность юнита вести огонь не меняется.

Встречается в команде actor_state.
