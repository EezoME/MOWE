Air Attack

Команда air_attack приказывает юниту атаковать с воздуха.

По строкам:

  • selector - указывает на юнита
  • drop - дополнительный параметр для селектора
  • enemies - указывает на врага
  • clone - клонирует юнита(наверное)
  • count - указывает число атак (наверное)
  • move_from_back - задаёт обратный порядок
  • force_placement - принудительная привязка?
  • random_weapon - случайный выбор оружия
  • waypoint - указывает на вейпоинт
  • waypoint_to_attack - указывает на вейпоинт для атаки
  • attack_altitude - задаёт высоту атаки
