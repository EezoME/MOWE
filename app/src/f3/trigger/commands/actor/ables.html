Ables

Команда ables позволяет добавить или убрать способности юнита, которые указываются в строке ables в настройках юнита справа.

По строкам:

  • selector - указывает на юнита
  • drop - дополнительный параметр для селектора
  • add - указывает способность, которую следует добавить
  • remove - указывает способность, которую следует убрать
