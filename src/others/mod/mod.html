Создание мода

Для того, чтобы было удобнее хранить вместе все элементы Ваших миссий, можно поместить их в мод.
Для начала, нужно в корне игры создать папку с именем 'mods'.
Далее в этой папке создать ещё одну с каким-либо названием, например 'my first mod'.
В папке мода создаём папки 'localization' и 'resource', а также файл 'mod.info' (редактируется любым текстовым редактором).
Этот файл имеет структуру, которую распознаёт игра. Обязательные параметры: name и desc.
ВАЖНО: значение параметра name должно совпадать с именем папки мода.

В папке 'localization' создаём подпапки как в директории игры. Тут будут хранится тексты.

В папке 'resource' также создаём дерево. Тут будут храниться все остальные ресурсы.

Далее приведён пример общй структуры мода. 

    {root}
        mods
            {mod}
                localization
                    mission
                        single
                            {campaign}
                resource
                    interface
                        text
                            mission
                                single
                                    {campaign}
                    map
                        single
                            {campaign}
                                {mission}
                    music
                    video
                        mission
                            single
                                {campaign}
                                    {mission}
                mod.info

Файлы в папках

Папка {campaign} в localization/mission/single и resource/interface/text/mission/single, как правило, содержит в себе следующие файлы:

  • {mission}.lng - стандартный файл с текстами миссии (может быть несколько)
  • campaign.lng - файл с описанием кампании
Файл campaign.lng имеет обязательные параметры title и name, где указывается подпись к кампании и её название соответственно.

Папка {mission} в resource/map/mission/single/{campaign}, как правило, содержит в себе следующие файлы:

  • {f3}.mi - файл миссии (создаётся редактором при сохранении миссии в РРМ)
  • map - файл карты (создаётся редактором при сохранении миссии в РРК)
  • map.grs - файл карты (создаётся редактором при сохранении миссии в РРК)
  • map.net - файл карты (создаётся редактором при сохранении миссии в РРК)
  • map.ply - файл карты (создаётся редактором при сохранении миссии в РРК)
  • minimap.jpg - изображение миникарты
  • preview.jpg - изображение экрана загрузки миссии

Примечания

  • {root} - корневая папка игры (например, моя ВТВ2:ЛП установлена по адресу 'E:\Games\VtV\menofwar')
  • {mod} - название папки мода
  • {campaign} - название папки кампании
  • {mission} - название папки миссии
  • {f3} - название файла миссии, которое Вы ему даёте при сохранении миссии в РРМ
Названия директорий кампании и миссии должны совпадать с аналогичными в папке {root}/resource/map/single/.
