Source (waypoint)

Строка source, тип lt2. При значении по-умолчанию (standart) открывается ряд строк, которые соответствуют селектору II типа.

При значении advanced ничего не открывается. Однако, если нажать правой кнопкой мыши по строке actors, откроется контекстное меню с пунктами выбора расширенной настройки.

Если выбрать + group откроются ещё 3 строки.

Если нажать правой кнопкой мыши на строку select откроется контекстное меню указания юнита по базовым параметрам.

Если нажать правой кнопкой мыши на строку include откроется контекстное меню указания юнита по дополнительным параметрам.

Если нажать правой кнопкой мыши на строку exclude откроется контекстное меню указания юнита по дополнительным параметрам.

Встречается в команде waypoint и селекторах всех типов.
