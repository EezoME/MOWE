Окно вызова функции (/НЗ)

Окно вызова функции (call function) позволяет узнать, какие функции доступны выделеному объекту.

Если у Вас есть информация, которая поможет закончить эту страницу, присылайте её мне (Дмитрию Чурикову). Я её размещу сдесь, естественно, указав источник (то есть Вас).

Наибольшее количество функций доступно для юнитов. В качестве примера разберём пехотинца.

Во вкладке AniPlay можно указать название анимации (name), скорость (speed) и "кость" объекта, к которому привязываемся??(bone_name).

Во вкладке AniStop можно указать только имя анимации, которую необходимо остановить.
Вкладка Clone пустая??.

Во вкладке Cover можно указать ранг??.

Во вкладке Freq моно указать тег.

Во вкладке MoveWp можно указать номер вейпоинта.

Во вкладке SetSpeed можно указать скорость в км/ч.
Вкладка brain/Debug пустая??.
