Mission Properties

Окно свойств помогает сделать миссию красивой и со всеми особенностями.

Группа show состоит из трёх строк: waypoint, zones и covers. Если эти строки принимают значение ИСТИНА, то в редакторе всегда будут отображаться вейпоинты, зоны и укрытия соответственно. Если же значание строк ЛОЖЬ, то эти элементы будут отображаться только на вкладках, где они редактируются (удобно при очень громоздких миссиях). На картинке все эти элементы будут отображаться в редакторе.

Группа player состоит из трёх строк: сама строка player, diplomacy и aliaces.

В строке player выбирается главный плеер, тот, кем будет управлять игрок.


Строка diplomacy открывает окно отношений между плеерами. Всего существует три вида отношений: союзник, враг и нейтральный. По-умоланию всех плеера между собою враги. Меняются отношения левой и правой кнопками мыши.


Строка aliaces открывает окно редактирования плееров. Тут можно задать число плееров (строка count), а также указать им вместо стандартного имени своё, которое лучше описывает суть/предназначение плеера.


Строка fog включает туман войны, но он почему то не работает.

Строка tasks открывает редактор заданий.

Строка reinforcements открывает редактор подкреплений.

Строка event открывает окно событий.


Строка vars открывает окно переменных.


Строка functions открывает окно функций.

Строка text открывает текстовый браузер

Строка breeds открывает редактор юнитов.

Строка environment открывает окно окрущающей среды

Строка modifiers открывает редактор модификаторов.

Строка entity_manager - булевская (тип lb).

При значении ИСТИНА открывает доп. строку count, в которой указывается коефициент быстроты исчезновения разрушеной техники.

Строка azimuth указывает начальную ориентацию для карты и юнитов на ней. Принимает числовое значения (по-умолчанию - 0), открывает окно выбора направления.

Строка map_orientation (тип lt2) указывает начальную ориентацию для карты.
