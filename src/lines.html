Типы строк

Я распределил все строки в командах на 10 типов:

lt1 (line type 1) - выпадающий список типа listbox. В списке, помимо прочих параметров, обязательно присутсвует параметр "-", который позволяет не выбирать ни одного из возможных параметров. Пример: строка state.

lt2 (line type 2) - выпадающий список типа listbox. В отличии от lt1, не имеет параметра "-", следовательно пользователь обязательно должен указать один из параметров. Пример: строка operation.

lt3 (line type 3) - выпадающий список типа listbox. В качестве параметров всегда принимает всего три значения: -, on, off. Параметр "-" не меняет значение, on - устанавливает, off - снимает. Пример: строка weapon_prepare.

lt6 (line type 6) - выпадающий список типа checkbox. Схожий с listbox, только в этом есть возможность выбрать несколько параметров. Возле каждого параметра стоит квадратик (если он закрашен, значит параметр выбран). В строке отображаются по очереди все выбранные параметры. Пример: строка drop.

lt7 (line type 7) - выпадающий список типа listbox. Во многом аналогичен lt1, только в этом типе значение можно вводить вручную. Наиболее известное примeнение - строка tag, поскольку даёт возможность приписать тег, которого у юнита нет сейчас, но он его может получить по ходу игры (например, explosion или captured).

lb (line boolean) - строка булевского типа, т.е. имеет только два возможных значения: ИСТИНА или ЛОЖЬ. Пример: строка ignore_captured_by_user.

ln (line numeric) - строка числового типа, т.е. в этой строке указывается число. Часто имеет значение по-умолчанию 0. Значение -1 означает бесконечное количество. Пример: строка amount.

lt (line text) - строка для ввода текстового значения. Пример: строка effect.

ls (line selection) - строка (с кнопочкой с троиточием) открывает другое окно для указания в нём нужных значений или параметров. Пример: строка selector.

lts (line text+selection) - строка для ввода текстовго значения с возможностью открыть другое окно для выбора значения вручную. По сути объеденяет свойства строк типа lt и ls. Пример: строка text.

Изменения:

  1. Тип lt5 удален из общего списка.
    lt5 (line type 5) - выпадающий список типа listbox. В качестве параметров всегда принимает всего три значения: -, disable, enable. Параметр "-" не меняет свойство, disable - деактивирует, enable - активирует. Пример: строка ai_move.

  2. Тип lt4 удален из общего списка.
    lt4 (line type 4) или lp (line player) - выпадающий список типа listbox, частный случай строки типа lt2. В качестве параметров всегда принимает список доступных плееров (обычно их 8: от player0 до player7). В редакторе всего две строки имеют такой тип: player и miner.
