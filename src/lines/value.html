Value

Строка value, тип lt7. Указывает второй операнд (число) для арфиметических операций. Список значений состоит из всех переменных в миссии. Специальная переменная players:count@ возвращает число плееров. Значение по-умолчанию приравнивается к нулю.

Встречается в командe set_i.
