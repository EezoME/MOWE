Текстовый браузер

Для того, чтобы текст отображался как диалог, его необходимо создать в текстовом браузере (text browser). Также, текстовый браузер (далее "ТБ") позволяет наглядно оценить текст на читабельность, наличие ошибок и т.д. Все диалоги и тексты бриффингов и эпилогов рекомендуется записывать в ТБ. Разработчики прописывают в ТБ даже тексты заданий и подкреплений.

Существует много способов открыть ТБ. Я открыл ТБ через F9 -> texts. Также, ТБ можно открыть в каждой команде, где могут использоваться его тексты.

Интерфейс ТБ следующий. Слева отображаются все созданные тексты. Три управляющие кнопки снизу позволяют Добавить, Удалить или Переименовать текст.
Большая область справа показывает текст целиком, а в строчке под ней вводится непосредственно сам текст.
Чтобы сохранить изменения, внесённые непосредственно в строку, необходимо нажать клавишу ENTER, иначе при клике где-либо (за пределами строчки) все изменения утратятся и придётся заново вводить текст.
После этого нажмите на кнопку Save и все изменения в тексте сохранятся. Кнопка Undo отменяет последние изменения (если Вы нажали ENTER).
Нажмите OК, чтобы выйти из ТБ, сохранив все изменения или Отмена, чтобы выйти без сохранения.

В ТБ также есть зарегистрированные имена текстов, которые отвечают за интерфейс миссии:
Текст с именем date указывает дату событий данной миссии (например, освобождение Николаева произошло 28 марта 1944 года). На скриншоте указана дата написания этой страницы.

Текст с именем place указывает место, где происходят действия (например, Николаев).

Текст с именем name указывает название миссии, которое отображается в меню кампании.

Текст с именем name_big указывает название миссии, которое отображается при загрузке миссии. Обычно name и name_big имеют одинаковый текст.

Текст с именем outro_completed содержит текст, который будет отображаться в итоговом окне Победы.

Текст с именем outro_failed содержит текст, который будет отображаться в итоговом окне Проигрыша.

Текстовый браузер можно открыть в следующий командах: talk (строка text), objective_progress (title), gameover (message), scenario (text), timer (title) а также через F9 в заданиях (desc) и подкреплениях (text, hint).
