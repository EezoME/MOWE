State (селектор)

Строка state, тип lt1. Указывает состояние юнита. Имеет 8 значений:

  • dead - мертвый
  • not dead - живой
  • operatable - действующий
  • not operatable - не действующий
  • moveable - движеспособный
  • not moveable - не движеспособный
  • inhabited - с экипажем
  • not inhabited - без экипажа

    • При значении по-умолчанию этот параметр не уточняется.

    Встречается в селекторах всех типов.
