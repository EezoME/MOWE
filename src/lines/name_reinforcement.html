Name (reinforcement)

Строка name, тип lt1. Указывает подкрепление, над которым необходимо произвести действие. Доступно только если в строке operation выбрано show или hide. В качестве параметров принимает список доступных подкреплений. При значении по-умолчанию подкрепление считается не заданным.

Встречается в командe reinforcement.
