If

Команда if вкладывает условие между командами, для более точного и удобного программирования миссии.
В эту команду можно вкладывать другие команды через контекстное меню. Вложеные команды выполняются если условие в строке type возвращает ИСТИНУ.

По строкам:

  • type - указывает на условие
