Actor Component

Команда actor_component позволяет выполнить действие над компонентом. Этой командой можно, например, сломать двигатель грузовику, сбить трак танку или починить его.

По строкам:

  • selector - указывает на юнита
  • drop - дополнительный параметр для селектора
  • component - указывает на компонент
  • state - указывает состояние, в которое переходит компонент
