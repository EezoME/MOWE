Emit

Команда emit высаживает пехоту с техники.

По строкам:

  • selector - указывает на юнита
  • drop - дополнительный параметр для селектора
  • waypoint - указывает на вейпоинт, к которому следует двигаться после покидания техники
  • crew - указывает на юнитов в технике
  • mode - дополнительный параметр для того, кому следует покинуть технику
  • count - максимальное число пехоты, которая покидает технику
