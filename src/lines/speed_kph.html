Speed kph

Строка speed_kph, тип ln, задает скорость движение юнита в км/ч. При вводе цифры, которая превишает максимальную скорость юнита, он будет двигаться с максимальной скоростью. Значение по-умолчанию для этого параметра равно 0, что означает, что юнит скорость свою не меняет.

Встречается в командe actor_state.
