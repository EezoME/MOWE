Throw Off

Команда throw_off бросает юнита на заданые высоту и длину в заданом направлении. (не проверял)
Например, при взрыве можно использовать эту команду для иммитации того, что юнита откинуло ударной волной.

По строкам:

  • selector - указывает на юнит
  • drop - дополнительный параметр для юнита
  • up - высота, на которую нужно подкинуть юнита
  • forward - расстояние, на которое необходимо откинуть юнита
  • turn - угол разворота
  • keep_orientation - сохранять направление юниту
  • die - умертвить?
