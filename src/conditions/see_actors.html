See Actors (условие)

Условие see_actors срабатывает, когда один заданый юнит видит другого заданого юнита.

По строкам:

  • mode - указывает на состояние условия
  • selector - указывает на один юнит
  • enemy - указывает на другой юнит
  • distance - указывает расстояние
  • detection - указывает на степень распознания
  • tag - при значении ИСТИНА открывают ещё две строки, которые указывают теги юнитов
