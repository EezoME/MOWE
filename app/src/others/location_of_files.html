Расположение файлов

Редактор позволяет добавлять в миссии свой медиаконтент.
Для того, чтобы он мог их найти, нужно поместить их в определённые директории.
Для аудиофайлов - это директория {root}/resource/music.
Для видеофайлов - это директория {root}/resource/video/mission/single/{campaign}/{mission}.
Тексты для миссий располагаются по адресу: {root}/resource/interface/text/mission/single/{campaign}/{mission}.

Примечания

  • {root} - корневая папка игры (например, моя ВТВ2:ЛП установлена по адресу 'E:\Games\VtV\menofwar')
  • {campaign} - название папки кампании
  • {mission} - название папки миссии
Названия директорий кампании и миссии должны совпадать с аналогичными в папке {root}/resource/map/single/.

Редартор поддерживает следующие расширения файлов:

  • Изображения
    • .jgp - пользовательский
    • .bmp - пользовательский, элементы интерфейса
    • .dds - пользовательский, текстуры
  • Аудио
    • .mp3 - музыка
    • .wav - звуки
  • Видео
    • .avi
