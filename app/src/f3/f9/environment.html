Окружающая среда (страница не закончена)

В редакторе есть возможность настоить погодные условия, освещённость и т.д. Всё это в совокупности называется окружающей средой (environment).

Справа находится список доступных сред.
В свойствах можно настроить следующие параметры:

  • light - свет
    • direction - направление
    • ambient - внешней среды
    • diffuse - рассеивания
    • specular - зеркальный
    • lightsIntensity - интенсивность источников света
    • night - ночь
    • night_vision - степень видимости ночью
    • color_saturation - насыщенность цвета (1 - обычная насыщенность)
    • humans_light - свет от человека?
      • user_ambient - цвет внешней серды для игрока
      • user_diffuse - цвет рассеивания для игрока
      • ally_ambient - цвет внешней серды для союзника
      • ally_diffuse - цвет рассеивания для союзника
      • enemy_ambient - цвет внешней серды для врага
      • enemy_diffuse - цвет рассеивания для врага
    • vehicles_light - огни транспортных средств
      • user_ambient - цвет внешней серды для игрока
      • user_diffuse - цвет рассеивания для игрока
      • ally_ambient - цвет внешней серды для союзника
      • ally_diffuse - цвет рассеивания для союзника
      • enemy_ambient - цвет внешней серды для врага
      • enemy_diffuse - цвет рассеивания для врага
  • shadow - тень
    • alpha - альфа? (главный наверное)??
    • color - цвет
  • wind - ветер
    • speed - скорость
    • amplitude - амплитуда
  • haze - туман (не "туман войны")
    • enable - включён
    • color - цвет
    • start - начало?
    • end - конец?
  • sky - небо
    • enable - включено
    • texture - текстура
    • size - размер (1 1 - обычный размер)
    • scrollSpeed - скорость прокрутки?
    • axis - ось
    • location - позиция
    • globalEnvironment - глобальная среда
  • cloudsonwater - облака, солнце и вода
    • enable - включено
    • texture - текстуры
    • size - размер
    • scrollSpeed - скорость прокрутки?
    • windSpeed - скорость ветра
  • weather - погода
    • enable - включено
    • density - плотность
    • fx - эффект (blizzard - метель, rain - дождь, rain_night - ночной дождь (темнее чем просто дождь), snow - снег, snow_flakes - хлопья снега)
    • sound - звук
  • thunder - гром
    • enable - включено
    • interval - интервал
    • color - цвет
    • sound - звук
  • video_overlay - наложение видео
    • on - включено
    • file - файл
    • color - цвет

Также открыть окно с окружающей средой можно нажав клавишу F6.
Установить или изменить окрущающую среду можно через команду environment.
