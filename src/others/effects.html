Эффекты

Эффекты (effects) позволяют объектам выполнить действие, которое обычным способом выполнить не получится.
Список известных мне эффектов приведён ниже:

  • open - открыться (для ворот)
  • close - закрыться (для ворот)
  • up - подняться вверх (например, для мишеней в тире)
  • down - опуститься вниз (например, для мишеней в тире)
  • start - начать действие (скрипт с парашютистами)
  • stop - прекратить действие (никогда не встречал)
  • explosion - взорваться (танк, цистерна, радар)
  • -

Эффект указывается в команде effect
