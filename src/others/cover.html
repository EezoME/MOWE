Cover

Иногда появляется необходимость направить юнита в конкретное укрытиe. Для этих (и, возможно, других) целей существует вкладка укрытие (cover).

В верхнем окне отображаются все доступные укрытия (те, что Вы создали или пометили). Для этого окна существует фильтр, который отбирает из всех укрытий только те, что нас интересуют. Фильтр может выбрать только изменённые, именованные, установленные вручную, с приоритетом, выключенные или именованные и установленные вручную.

А теперь по подробнее. Изменённые (modified) - это те, свойства которых были изменены.
Именованые (named) - это те, у которых есть имя. В игре практически каждый объект имеет множество укрытий вокруг себя, но они все не имеют имени и не отображаются в этом списке. Для того, чтобы внести их в список, им надо задать имя, т.е. пометить.
Установленные вручную (manual) - это укрытия, добавленные через кнопку Add. Такое укрытие можно установить в любом месте на карте вне зависимости есть ли рядом какой-либо объект.
С приоритетом (priority) - укрытия, у которых с строке приоритета значение больше нуля.
Выключенные (turn off) - все неактивированные на данный момент укрытия.

Идём дальше. Кнопка Add позволяет добавить укрытие вручную (manual). После нажатия на неё, возле курсора появится метка, похожая на вейпоинт.

В окне Properties можно настроить свойства укрытия.

Строка type укажет тип укрытия и поведение юнита в нём. Имеет целый список значений: смотреть влево, смотреть вправо, смотреть в обе стороны, укрытие снайпера, присесть, скрытно стоять, скрытно прижаться, open_lie??, стоя в кустах, вприсед в кустах, лёжа в кустах, лёжа на ландшафте.

Name - имя укрытия.
Off - деактивирует укрытие (солдат в него никогда не пойдёт).
Priority - приоритет укрытия над другими.
Position - координаты укрытия.
