Actor State

Команда actor_state позволяет изменить первичные параметры юнита. Большинство настроек, указанных здесь, можно указать в настройках юнита справа.

А теперь по строкам:

  • selector - указывает на юнита
  • drop - дополнительный параметр для селектора
  • control - указывает, кому подконтрольный данный юнит (ИИ или игроку)
  • fire_mode - режим стрельбы
  • move_mode - режим передвижения
  • move_radius - радиус передвижения
  • weapon_prepare - достать/спрятать оружие
  • ai_move - разрешает ИИ управлять юнитом, подконтрольным игроку
  • movement - настройки передвижения
    • speed - скорость по качественной шкале
    • kind - вид передвижения
    • speed_kph - скорость по количественной шкале (км/ч)
    • max_speed_kph - максимальная скорость (км/ч)
    • type - тип передвижения
    • forward и backward - слияет на склонность юнита занимать передний или задний фланги
  • human - параметры, примеными только для пехоты
    • health - запас здоровья (100 - по-умолчанию максимально возможное число единиц здоровья для пехотинца)
    • energy - запас энергии (100 - по-умолчанию максимально возможное число единиц энергии для пехотинца)
  • ai - настройки для ИИ
    • no_advance - юнит не будет сам наступать
    • no_retreat - юнит не будет сам отступать
    • advance_ratio - коэфициент наступления
    • retreat_ratio - коэфициент отступления
    • follow_commander - следовать ли за коммандиром или оставить его в покое
    • make_enemies_forget - забывать про врагов, если они вне поля зрения
  • cannon - параметры для орудия
  • unmasking - способы раскрытия юнита
