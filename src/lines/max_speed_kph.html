Max Speed kph

Строка max_speed_kph, тип ln, задает максимальную скорость движение юнита в км/ч. При вводе цифры, которая превишает максимально разрешённую скорость юнита, будет установлена самая большая максимально разрешённая скрость. Значение по-умолчанию для этого параметра равно 0, что означает, что юнит свою максимальную скорость не меняет.

Встречается в командe actor_state.
