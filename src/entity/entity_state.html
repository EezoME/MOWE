Entity State

Команда entity_state изменяет некоторые свойства объекта.

По строкам:

  • selector - указывает на объект
  • tag_add - тег для добавление
  • tag_remove - тег для удаления
  • inactive - деактивирует объект
  • impregnability - устанавливает степень уязвимости
  • examined - указывает изучен ли объект
  • discovered - указывает обнаружен ли объект
