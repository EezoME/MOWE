Action

Команда action побуждает юнита на действие. Действие может быть выполненым юнитом как по отошению к себе самому так и по отношению к другому юниту.

А теперь по строкам:

  • selector - указывает на юнита
  • drop - дополнительный параметр для селектора
  • action - указывает действия
  • target - цель, по отношению к которой нужно выполнить действие (например, грузовик для осмотра или вражеский пехотиненц для атаки)
  • volume - предназначение этой строки для меня пока остаётся не понятным
  • waypoint - указывает целевой вейпоинт
