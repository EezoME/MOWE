Talk

Команда talk выводит сообщение в виде фразы от юнита (диалог).

По строкам:

  • selector - указывает на юнита
  • drop - дополнительный параметр для селектора
  • say - указывает на сопровождающий звуковой эффект в виде фразы
  • text - указывает на текст в текстовом редакторе
  • clear_queue - обеспечивает строгую очерёдность вывода сообщений
