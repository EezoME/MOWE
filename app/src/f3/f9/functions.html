Функции

Иногда нам необходимо выполнять одни и те же действия несколько раз. В таких случаях пригодятся функции (functions). Функция в редакторе - это порядок команд, которые выполняют определённую логику миссии.

Для того, чтобы создать функцию, необходимо в окне, открывающимся клавишей F9, открыть строку functions. Откроется окно функций.

Для добавления новой функции нажмите клавишу INSERT или нажмите ПКМ и выберете пункт Add..
После этого откроется окно ввода названия функции.

Новая функция добавится в конец списка.

Также функции можно удалять, переименовывать и копировать (удобно при схожих функциях).

Чтобы наполнить функцию командами, нажмите на троеточие в правой части строки или двойной клик по строке. Откроется окно, в котором можно добавить команды.

Для добавления команды нажмите клавишу INSERT или нажмите ПКМ и выберете пункт Add... Откроется стандартное окно выбора команд.

Вызвать функцию можно через команду call. В строке function содержится список всех доступных функций.
