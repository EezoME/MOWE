Trigger On Cancel

Строка trigger_on_cancel (аналогичен строке name с команды trigger), тип ls. Указывает триггер, который должен сработать при прерывании показа ролика. Открывает окно триггеров. При значении по-умолчанию этот параметр не задаётся.

Встречается в команде scenario.
