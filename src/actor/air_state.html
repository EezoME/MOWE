Air State

Команда air_state изменяет параметры юнита, передвигающегося по воздуху.

По строкам:

  • selector - указывает на юнита
  • drop - дополнительный параметр для селектора
  • ground - по всей видимости указывает на то, находится ли юнит на земле или в воздухе
  • altitude - задаёт новый параметр высоты
