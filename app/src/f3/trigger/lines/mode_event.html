Mode (event)

Строка mode, тип lt2. Задаёт действие для события, указаного в строке event. При значении reset событие сбрасывается, set - срабатывает, toggle - переключается на противоположное (если было сброшено или ещё не сработало - срабатывает, если сработало - сбрасывает). Значение по-умолчанию - set.

Встречается в команде event.
