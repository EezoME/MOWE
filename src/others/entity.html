Entity

Вкладка entity есть и в РРК и в РРМ. В ней можно поставить объект на карту. Также, при выделении объекта показываются его свойства.

Разберёмся по строкам (в качестве примера был выделен русский офицер):

  • mid - мид объекта
  • desc - описание объекта (которое видно при наведении в РПС)
  • ables - способности юнита. Подробно список описан тут
  • player - плеер юнита
  • mask_player - замаскированый плеер (этот плеер может быть вражеским, тогда враги будут считать его как своего пока не раскроют)
  • unmasking - кто может раскрыть юнита. Подробно список описат тут
  • brain - мозги юнита (описывают его поведение)
    • fix_inventory - фиксированный инвентарь
    • control - кем управляется (см. строка control )
    • fire_mode - режим стрельбы (см. строка fire_mode )
    • move_mode - режим передвижения (см. строка move_mode)
    • move_radius - радиус передвижения (см. строка move_radius)
    • ai_move - управление компьютером (см. строка ai_move)
    • no_advance - не наступать (см. строка no_advance)
    • no_retreat - не отступать (см. строка no_retreat)
    • advance_ratio - рейтинг наступления (см. строка advance_ratio)
    • retreat_ratio - рейтинг отступления (см. строка retreat_ratio)
  • name - имя объекта
  • slot_id - слот объкта
  • constrain - привязка (обычно к земле)
  • position - позиция
    • x - первая координата
    • y - вторая координата
    • z - третья координата
  • xform - дополнительные оси
    • zL - угол по горизонтальной площади
    • y - угол по вертикальной площади
    • zW - вращение на заданый угол по горизонтальной площади от текущего значения (при этом значение в zL не меняется)
    • scale - масштабирование объекта
  • view - вид
    • stamp - печатки (если можно так выразиться)
      • decal - отображение
      • order - приоритет отображения
    • shadow - тень
    • textmods - текстуры
    • opacity - степень темноты
    • no_ground - без привязки?
    • color - цвет (white - обычный)
    • saturation - насыщенность
    • ingore_saturation - игнорировать строку saturation
    • z_clip_visibility - если ИСТИНА: при близости камеры объект становится полупрозрачным
  • fogable - склонность к туману войны?
  • storeable - возможность переносить объект в другую миссию (наверное)
  • mass - масса
  • discovered - открыт ли объект (см. строку discovered )
  • impregnability - неуязвимость (см. строку impregnability )
  • tags - теги объекта
  • simulation_ables - симуляция способностей
  • vitality - стойкость
    • hmax - максимальный запас здоровья
    • hcurrect - текущий запас здоровья
    • energy - запас энергии
  • stuff - ??
  • breed - брид
  • skin - скин
  • name - имя персонажа
    • first - Фамилия
    • last - Имя
    • unique - уникальные Имя и Фамилия

Спавнить объекты во время миссии можно через команду spawn.
