Time

Строка time, тип ln, указывает время в секундах, которое нужно для команды. В команде delay - это время задержки, actor_fire - общее время атаки, wait - время ожидания, fade - время действия затемнения, timer - время счёта таймера. Значение по-умолчанию для каждой комманды разное: для delay, actor_fire, wait - 0, для fade - 1, для timer - пустое поле, что эквивалентно 0.

Встречается в командах delay, actor_fire, wait, fade, timer.
