Operation (player)

Строка operation, тип lt2. Указывает операцию по смене плеера. Имеет два значения: set - задаёт юниту новый плеер (маска плеера меняется автоматически), mask_set - задаёт юниту новую маску плеера (при этом значение плеера не меняется). Значение по-умолчанию - set.

Встречается в командe player.
