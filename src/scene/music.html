Music

Команда music добавляет фоновую музыку в миссию.

Файлы для вопроизведения необходимо помещать в папку music в директории игры.

По строкам:

  • start - указывает на файл, который воспроизводится
  • stop - указывает на файл, который останавливается
  • stop_all - останавливает воспроизведение всех текущих файлов
  • volume - устанавливает громкость музыки
  • fade - устанавливает затемнение (скорее всего)
  • loop - воспроизводит файл в строке start по кругу, пока тот не будет остановлен этой же командой
