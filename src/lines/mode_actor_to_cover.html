Mode (actor_to_cover)

Строка mode, тип lt2. Указывает дополнительный параметр для укрытия. При значении best - выбирается лучшее из укрытий, random - случайный выбор, density - укрытие с найбольшим числом юнитов вокруг (возможно имеются ввиду уже занятые укрытия). Значение по-умолчанию - best.

Встречается в команде actor_to_cover.
