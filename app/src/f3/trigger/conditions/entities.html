Entities (условие)

Условие entities срабатывает, когда условие с объектами, заданое строками стандартного селектора, совпадает с реальными объектами в игровом процессе.
Проще говоря, проверяет на наличие тех или иных объектов на карте с задаными параметрами.

По строкам:

  • mode - указывает на состояние условия
  • source - указывает параметр селектора
  • ignore_captured_by_user - игнорирует захваченого игроком
  • mids - указывает на мид
  • tag - указывает на тег
  • zone - указывает на зону
  • player - указывает на плеер
  • type - указывает на тип юнита
  • state - указывает на состояние юнита
  • count - сравнивает текущее значение заданого селектора со строкой ниже заданой булевской операцией
  • value - указывает число для сравнения
