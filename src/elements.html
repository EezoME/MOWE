Элементы миссии

Объект

Объект - любое тело, которое может поставиться на карту через вкладку entity.

Юнит

Юнит - объект, которым непосредственно может управлять игрок (пехота, техника, орудия и т.д.).

Мид

Мид - уникальный номер юнита, присваемый ему при создании. Указать мид - самый точный способ указать на юнита.
Отображение мидов включается сочетанием клавиш Alt+I.

Тег

Тег - метка, присваимая объекту для его принадлежности к некой группе. Например, если Вам нужно указать несколько юнитов, проще будет поставить им общий тег, нежели перечислять их по мидам. А в случае объектов, не имеющих мида, тег - единственный способ обозначить объект.
Также теги могут присваиватся по ходу игры. Например, при уничтожении объекта ему может быть присвоен тег crashed (разрушен для не юнитов), destroyed (разрушен для юнитов) или burned (сожжён).
Такой тег как hidden скрывает объекты при запуске сценария и раскрывает их только когда они начинают действовать через команды (например, юнит был скрыт и по комманде начинает идти по вейпоинтам).
Отображение тегов включается сочетанием клавиш Alt+T.
Перечень зарезервированых тегов.

Вейпоинт

Вейпоинт - точка с уникальным номером, которая может быть частью пути. Если на вейпоинте есть команды, он окрашивается в красный оттенок.

Путь

Путь - массив вейпоинтов, связаных между собой в определённом порядке.

Зона

Зона - область игрового пространства. Обычно в ней должно происходить что-то интересное раз уж нам понадобилось выделить именно этот участок поверхности.

Плеер

Плеер - сторона, которой пренадлежит тот или иной юнит. Главным плеером считается тот, которым играет игрок. Плеера могут быть друг другу врагами, союзниками или нейтральными. Цвет текущего плеера указывается квадратиком в верхнем левом углу экрана.

Ролик

Ролик - видеофайл или видео, сделанное в редакторе (во вкладке camera).
