Add


Строка add, тип lt6, добавляет свойства в настройку ables. Может принимать следующие параметры:

  • crew - техника может вмещать экипаж
  • ally_crew - техника также может вмещать союзных юнитов
  • any_crew - техника может вмещать любого юнита
  • repair - техника имеет свойство чиниться, если это возможно
  • repaired - техника считается починеной
  • talk - солдат может говорить (имеется ввиду, что его иконка может появляться в диалогах)
  • select - юнит может быть выбран
  • fight - юнит может сражаться
  • personage - действующее лицо/особа ?
  • recruit - солдат может быть призван союзником
  • unmask - солдат может быть раскрыт
  • emit_on_attack - высадка всех солдат из техники/орудия при атаке врага
  • emit_passangers_on_enemy - высадка солдат-пассажиров из техники при обнаружении врага
  • emit_crew_on_injure - высадка экипажа при ранении (используется для открытой техники и орудий)
  • emit_on_mandie - высадка всех солдат из техники/орудия при смерти одного из них
  • burning - солдат бегает, словно горит / техника считается объятой пламенем
  • sinking - солдат падает, словно мёртв
  • exploded - техника считаеся взорваной

При значении по-умолчанию никаких свойств не добавляется.

Встречается в команде ables.
