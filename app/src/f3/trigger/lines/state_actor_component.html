State (actor_component)

Строка state, тип lt1. Указывает на состояние компонента. Имеет 3 значения:

  • work - исправен
  • broken - сломан, но можно починить
  • destroyed - разрушен (починить нельзя)
При значении по-умолчанию этот параметр не меняется.

Встречается в команде actor_component.
