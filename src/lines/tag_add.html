Tag Add

Строка tag_add (во многом схожен со строкой tag в селекторах), тип lt7. В качестве параметров принимает список тегов (также тег можно прописать вручную). Добавляет заданый тег заданому юниту. При значении по-умолчанию список тегов у юнита не меняется.

Встречается в команде entity_state.
