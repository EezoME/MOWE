Type (movement)

Строка type, тип lt1, задаёт тип движения для юнита. При значении normal юнит движется обычно, crawl - ползёт, squat - приседает, swim - плывёт. При значении по-умолчанию ("none" или "-") этот параметр не меняется.

Встречается в командах actor_state, squad_state.
