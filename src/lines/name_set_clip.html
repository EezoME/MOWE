Name (set_clip)

Строка name_set_clip, тип lt1. Указывает клип (рамку), которую следует применить. В качестве параметров принимает список доступных клипов. Параметр [auto] присутствует по-умолчанию, поскольку такой клип создаётся автоматически и охватывает всю карту. При значении по-умолчанию этот параметр не меняется.

Встречается в командe set_clip.
