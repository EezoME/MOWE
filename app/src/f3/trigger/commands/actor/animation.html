Animation (команда)

Команда animation задаёт параметры для анимации. В отличии от команды animate тут задаётся анимация только для юнитов.

По строкам:

  • selector - указывает на юнита
  • drop - дополнительный параметр для селектора
  • collage - указывает название анимации, которое можно узнать с окна вызова анимации
  • speed - скорость анимации
  • circledelay - задержка между повторами
  • delay - начальная задержка (наверное)
  • totalTime - общее время выполенения анимации
  • flags - задаёт дополнительные флаги для анимации
