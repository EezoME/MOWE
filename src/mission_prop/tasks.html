Задания миссии

Чтобы игроку было интересней играть, необходимо задать ему цель. Для этого создаются задания.

Для того, чтобы создать такое задание, необходимо нажать клавишу F9 и в строке tasks открыть редактор заданий. C начала он будет пустым.

Для добавления нового задания нажмите клавишу INSERT или нажмите ПКМ и выберете пункт Add..
После этого откроется окно ввода названия задания (не путать с текстом задания!).

После ввода названия в окно редактора заданий добавятся строки группы, которая называется как Ваше задание.

Далее по строкам. В строке desc указывается текст задания.
Строка primary булевская и при значении ИСТИНА данное задание будет отображаться в списке основных в записной книжке, при значении ЛОЖЬ - в списке дополнительных.
Строка state указывает начальный статус задания. Имеет 3 значения: hidden - задание не показывается в записной книжке при запуске миссии, completed - задание считается сразу выполненым, failed - задание считается сразу проваленым (в обоих случаях не появляется надписи Задание выполнено или Задание провалено при запуске миссии).

В строке point указывается тег указательной точки, которая указывает на объект задания. Сам объект-точка находится по адресу service/pointer/pointer или pointer_small. Объект-точка по-умолчанию имеет тег pointer, что скрывает её, если задание, где она не используется, не активно (не выделено).

В строке team указывается команда, которой может быть доступно данное задание (применяется в мультиплеере).

Внедрить задание в игру можно через команду task.
