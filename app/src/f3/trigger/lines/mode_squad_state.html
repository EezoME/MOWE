Mode (squad_state)

Строка mode (аналог speed), тип lt1. Задаёт параметр скорости для юнита. При значении slow юнит движется медленно, normal - с обычной скоростью, fast - быстро, walk - шагом, bow - пригнувшись, assault - как при штурме, spot - как при обнаружении противника, sneak - крадётся. При значении по-умолчанию этот параметр не меняется.

Встречается в команде squad_state.
